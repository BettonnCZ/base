# setup_tools_createNewWorkspace

A simple website project built with Vite and vanilla JavaScript.

## Getting Started

1. Install Node.js from https://nodejs.org/

2. Install dependencies: `npm install`

3. Start the development server: `npm run dev`

4. Open http://localhost:5173 in your browser

## Build

`npm run build`

This creates a `dist` folder with production files.

## Deploy

Upload the `dist` folder to any static hosting service like:

- GitHub Pages

- Netlify (drag & drop `dist`)

- Vercel

- Firebase Hosting

## Features

- Dark futuristic theme

- Text inputs for title, summary, details

- 3 image upload windows (double-click to upload)

- Save/Load data to browser localStorage

- Export data as JSON
